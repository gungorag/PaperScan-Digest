# 🔬 PaperScan

A weekly digest of AI/ML papers in ophthalmology and neuro-ophthalmology, pulled from **arXiv**, **PubMed**, and **Semantic Scholar** — deduplicated and delivered to your inbox.

---

## What it does

- Searches the last **7 days** of publications across 3 sources
- Matches papers combining **medical topics** (fundus, OCT, OCTA, retina, visual field, MRI, neuro-ophthalmology…) with **AI methods** (deep learning, segmentation, classification, detection…)
- Deduplicates by DOI across sources
- Skips papers already sent in a previous run
- Sends a clean **HTML email digest** via Gmail

---

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/YOUR_USERNAME/paperscan.git
cd paperscan
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Configure credentials

```bash
cp env.example .env
```

Edit `.env` and fill in:

```
GMAIL_SENDER=email_address
GMAIL_APP_PASSWORD=your_16_char_app_password
GMAIL_RECIPIENT=email_address
```

> **How to get a Gmail App Password:**
> 1. Go to [myaccount.google.com/security](https://myaccount.google.com/security) → enable 2-Step Verification
> 2. Go to [myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords)
> 3. Create a new app password (name it "ophtho-tracker")
> 4. Copy the 16-character password into `.env`

### 4. (Optional) Customize keywords

Edit `config.yaml` to add/remove medical topics or AI methods.

## Usage

### Test without sending email

```bash
python tracker.py --dry-run
```

This prints all found papers to the terminal — nothing is sent and the cache is not updated.

### Run normally (sends email)

```bash
python tracker.py
```

### Schedule weekly (optional, on Linux/macOS)

```bash
crontab -e
```

Add this line to run every Monday at 8am:

```
0 8 * * 1 cd /path/to/paperscan && python tracker.py
```

## Project structure

```
paperscan/
├── env.example        # credentials template (never commit .env)
├── .gitignore         # excludes .env, cache, logs, run.sh
├── README.md
├── requirements.txt
├── config.yaml        # keywords + settings
├── tracker.py         # main script
├── schedule.sh        # sets up anacron (run once)
├── unschedule.sh      # removes the schedule
├── run.sh             # auto-generated by schedule.sh (gitignored)
├── sent_cache.json    # auto-created, tracks sent paper IDs (gitignored)
└── logs/
    └── anacron.log    # auto-created run logs (gitignored)
```

## Notes

- `sent_cache.json` is gitignored — it lives only on your machine
- Running twice in the same week won't re-send already-delivered papers
- Use `--dry-run` to preview before sending
